</h4>

---

## Installation

To install it, you must run in your terminal:

```sh
$ git clone https://github.com/daniloalbuqrque/port-scanner
```

## Usage

```sh
$ python3 nerg.py [hostname or target IP]
```
e.g.:
```sh
$ python3 nerg.py localhost
```
```sh
$ python3 nerg.py 127.0.0.1
```
